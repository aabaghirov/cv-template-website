# CV Template Website

This project is a simple web application that serves a resume using Flask. The application is structured to include a server script, an HTML template for the resume, and a CSS file for styling.

## Project Structure

```
cv-template-website
├── server.py          # Main server script
├── templates
│   └── template.html  # HTML template for the CV
├── static
│   └── format.css     # CSS styles for the CV
└── README.md          # Project documentation
```

## Setup Instructions

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd cv-template-website
   ```

2. **Install dependencies**:
   Make sure you have Python and pip installed. Then, install Flask:
   ```
   pip install Flask
   ```

3. **Run the server**:
   Execute the following command to start the server:
   ```
   python server.py
   ```

4. **Access the resume**:
   Open your web browser and navigate to `http://localhost/resume` to view the resume.

## Usage

- Modify the `templates/template.html` file to update the content of your resume.
- Customize the `static/format.css` file to change the styling of the resume.
